## 🔴 Red Rooms (Sub-Level 0)

Red Rooms use the same procedural generation system as Level 0, since they are essentially a variation of the same environment with a different **visual and audio theme**.

The underlying systems remain identical:
- Chunk-based procedural generation  
- Weighted randomness  
- Biome logic and streaming system  
- Persistent world state  

The main differences are the **color palette, lighting, and ambient sound design**, which create a more unsettling atmosphere.
